## Beginner React Workshop - Hack The Valley III

### How to run this
- Install [Node.js](https://nodejs.org/en/)
- Clone the repo
- Open up your favourite terminal, go to the folder, and run ```npm ci```
- Run ```npm start```
- Enjoy!
